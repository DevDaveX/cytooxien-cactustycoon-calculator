## 📖 Zweck dieses Repositories

Dieses Projekt wird als **Open-Source-Version** veröffentlicht, um vollständige Transparenz über den enthaltenen Code zu gewährleisten.

Ziel ist es:

- 🔍 Nachvollziehbar zu machen, dass **keine Schadsoftware enthalten ist**
- 🛡️ Vertrauen durch offenen Code zu schaffen
- 👨‍💻 Einblick in die technische Umsetzung zu geben

> Dieses Repository ist **nicht primär für produktiven Einsatz oder Self-Hosting gedacht**, sondern dient der Einsicht und Analyse.
